

## Description

[Jexia](https://www.jexia.com/en/) Demo with nestjs.

## Installation

```bash
$ yarn install
```

## Running the server

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
